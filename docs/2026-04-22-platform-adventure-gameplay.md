# Jump and Say: 平台探险模式 (Platform Adventure) 策划文档

> **版本:** 1.0
> **日期:** 2026-04-22
> **核心机制:** 慢动作声控魔法 + 体感闯关 + 独立奖励房

---

## 1. 游戏愿景
为 3-6 岁儿童打造一个“身临其境”的马里奥式英语启蒙世界。通过原地跑步和起跳控制角色，利用“声控魔法”击败单词怪，在充满成就感的反馈中完成英语口语和听力的练习。

## 2. 操作映射 (Motion Control)

| 动作 | 身体姿态 (MediaPipe) | 游戏反馈 |
| :--- | :--- | :--- |
| **向右跑** | 原地小跑 (监测躯干/腿部抖动) | 角色持续向右移动 |
| **跳跃** | 原地起跳 (监测重心快速上升) | 角色向上跳跃 (用于躲避障碍或顶方块) |
| **静止/减速** | 保持站立不动 | 角色逐渐减速停止 |

## 3. 核心机制：声控魔法 (Round 1 融合)

不再是简单的“撞怪减血”，而是将英语跟读转化为一种“魔法攻击”。

### 3.1 怪物光环与子弹时间
- **触发**：当角色进入单词怪 (如 `slime_normal.svg`) 的感应范围 (光环) 时。
- **效果**：游戏进入**慢动作 (Bullet Time)**，背景音效变缓，怪物头上浮现单词卡片。
- **交互**：
    - 屏幕中央弹出麦克风图标，播放单词发音。
    - 孩子进行跟读录音。

### 3.2 魔法反馈 (Pronunciation Result)
- **成功 (发音准确)**：
    - 角色挥动魔杖 (播放 `super.mp3`)。
    - 怪物被击败并**转化**为奖励：金币、弹簧 (`spring.svg`) 或护盾。
    - 恢复正常流速，角色获得加速冲刺。
- **失败 (发音不准/超时)**：
    - 怪物加速冲向角色。
    - 孩子需要快速**起跳**躲避。如果碰撞，损失一颗心。

## 4. 独立挑战房：宝藏密室 (Round 2)

### 4.1 进入机制
- 在关卡 50% 或末尾处设置带锁大门 (`lock_yellow.svg`)。
- 只有通过前面声控魔法收集到钥匙 (`key_yellow.svg`) 才能进入。

### 4.2 顶方块答题 (Quiz)
- **场景**：进入明亮的室内场景。
- **规则**：屏幕上浮现 3 个悬浮方块 (`block_exclamation.svg`)，每个方块贴有不同的图片。
- **题目**：播放听力题，“Where is the Apple?”。
- **交互**：孩子控制角色跑到对应方块下方，**原地起跳顶起方块**。
- **奖励**：答对则方块爆出大量金币 (`coin_gold.svg`)，并播放赞美音效。

## 5. 生命值与结算系统

### 5.1 生命值 (HUD)
- **UI**：左上角显示 3 颗心 (`hud_heart.svg`)。
- **损失**：触碰尖刺 (`spikes.svg`) 或未躲过怪物的碰撞。
- **死亡处理**：心耗尽后不直接 Game Over，而是后退到上一个检查点 (Checkpoint)，保持非挫败感体验。

### 5.2 关卡奖励
- **终点旗帜**：抓到 `flag_red_a.svg` 完成关卡。
- **星级评估**：根据跟读准确率和挑战房表现给予 1-3 星。
- **皮肤奖励**：3 星可解锁新角色色彩 (如 `character_pink`) 或头盔。

## 6. 五大主题关卡

1. **草地冒险 (Grassland)**: 教学关，地型平坦。
2. **沙漠遗迹 (Desert)**: 增加仙人掌障碍，学习食物词汇。
3. **冰雪世界 (Snow)**: 地面湿滑增加操控感，学习动作词汇。
4. **地底洞穴 (Cave)**: 增加梯子和火把，增加纵向移动空间。
5. **紫色幻境 (Purple)**: 终极关，云朵平台 + 最终 Boss (巨大史莱姆)。
