  方案一：单词大冒险模式（类马里奥关卡制）

  🎯 核心玩法

  将现有的跳跃+发音机制与横版闯关结合：

  第一关：草原入门 (Grass主题)
  玩家需要：
  1. 控制角色左右移动、跳跃躲避史莱姆
  2. 收集金币获得积分
  3. 遇到"单词方块"时必须正确发音才能通过
  4. 到达终点旗帜完成关卡

  难度：⭐
  敌人：史莱姆（慢速移动）、蜜蜂（空中巡逻）
  教学：基本跳跃+简单单词（cat、dog）

  第二关：沙漠迷宫 (Sand主题)
  新增机制：
  - 流沙陷阱（踩到会缓慢下沉，需要快速发音"help"自救）
  - 仙人掌障碍（不能触碰，需要跳过）
  - 钥匙门系统（收集正确发音单词获得钥匙）

  难度：⭐⭐
  敌人：老鼠（快速移动）、蜗牛（隐藏在壳里）
  教学：复合词、简单句型

  第三关：冰雪世界 (Snow主题)
  新增机制：
  - 冰滑路面（移动惯性增大）
  - 雪花遮挡视野（需要发音"clear"清除）
  - 冰块可被推动（解谜元素）

  难度：⭐⭐⭐
  敌人：冰鱼（水下攻击）
  教学：带"雪"的单词、冬天相关词汇

  第四关：火焰城堡 (Stone/Purple主题)
  新增机制：
  - 熔岩地形（掉落直接死亡）
  - 火焰史莱姆（需要发音"water"才能消灭）
  - 多层地图（需要攀爬梯子）

  难度：⭐⭐⭐⭐

  方案二：单词竞速模式（街机风）

  🎯 核心玩法

  无限跑酷 + 即时发音

  设计概念：
  - 角色自动向前跑动
  - 玩家通过左右倾斜/跳跃控制躲避障碍
  - 屏幕出现单词卡片时，必须在3秒内正确发音
  - 发音成功：获得加速、护盾、双倍积分
  - 发音失败：速度减慢、失去护盾

  三种难度：
  1. 慢速：单词停留5秒，简单词汇
  2. 中速：单词停留3秒，中等词汇
  3. 快速：单词停留2秒，复杂词汇+句子

  敌人动态生成：
  - 前方出现史莱姆 → 发音对应单词消灭它
  - 蜜蜂飞来 → 发音"swat"拍走
  - 地面陷阱 → 发音"jump"跳过

  // 史莱姆AI示例
  class SlimeEnemy {
    patrol(startX: number, endX: number) {
      // 左右巡逻
      if (this.x <= startX) this.direction = 1;
      if (this.x >= endX) this.direction = -1;
      this.setVelocityX(this.speed * this.direction);
    }

    onPlayerCollision(player: Player) {
      // 碰撞检测：玩家从上方跳到敌人头上 → 消灭敌人
      if (player.y < this.y - 10) {
        this.destroy();
        player.bounce(); // 弹跳
      } else {
        player.takeDamage(); // 扣血
      }
    }
  }
